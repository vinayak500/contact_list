## Technologies Stack:

### Components
* HTML,CSS,Javascript
* Node.js,MongoDB,Express.js



## How to Install

* Clone the project onto your local machine.
* Then cd contact_list
* npm install
* npm start
* Visit your app at http://localhost:8000.
